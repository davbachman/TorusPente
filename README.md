Web link: https://davbachman.github.io/TorusPente/

Instructions for use:
1. Open the web link above.
2. Double-click a board region to place a stone.
3. Player 1 places white stones, then Player 2 places black stones, alternating turns.
4. Drag left or right with the left mouse button to rotate the torus around its center axis.
5. Drag up or down with the left mouse button to rotate cell positions around the tube cross-section.
6. The first player to make 5 in a row (horizontal, vertical, or diagonal, with wrap-around) wins.
7. When a win occurs, the winning five cells are highlighted; click Restart Game to play again.
